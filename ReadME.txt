Read Me:

To run the program: 

python ./searchMaster.py

please make sure all the test files are in the same directory.
The program will search each query by trie, out put the result to a file called trieOutput.txt .
Then it will start searching by edit distance, out put the result to another file editDistanceOutput.txt.
Trie dictionary is also printed out to a file called trie.txt.
